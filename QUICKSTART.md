## ğŸ”§ Troubleshooting

### Port Already in Use
If port 3000 or 8000 is already in use:
- **Backend**: Edit `start.sh` and change `--port 8000` to another port
- **Frontend**: Edit `frontend/vite.config.ts` and change the port

### OpenAI API Errors
- Make sure your API key is correct in `backend/.env`
- Check your OpenAI account has credits
- Verify the API key has proper permissions

### Database Issues
- Delete `backend/fraud_monitoring.db` and run `python backend/setup.py` again

### Dependencies Not Installing
- Make sure Python 3.9+ and Node.js 18+ are installed
- Try running the setup steps manually (see README.md)
